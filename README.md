This project predicts future sales using Machine Learning techniques.

## Dataset
- SuperStore Orders Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Machine Learning

## Steps
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Sales Prediction

## Output
Machine Learning model is used to forecast sales based on historical data.
